hexo搭建个人博客

1. 官网安装node.js

安装成功后，进行验证

node -v
npm -v

2. 安装镜像

npm install -g cnpm -registry=https://registry.npm.taobao.org
  1. node.js npm install -g 全局安装和非全局安装的区别
  2. registry： 指的是镜像源指向XXX镜像

安装成功后，校验

cnpm
cnpm -v

3. 安装hexo

cnpm install -g hexo-cli

验证

hexo -v

4. 初始化和启动hexo

mkdir blog
cd blog
sudo hexo init
ls -l #查看生成的文件
hexo s   #启动hexo

5. 生成一篇博客文章

hexo n "我的第一篇博客文章"   #生成一篇博客文章

6. 编写/修改文章

vim 我的第一篇博客文章.md

markdown格式进行编写

hexo clean   #清理
hexo g   #生成
hexo s   #启动

7. 部署博客到github上

用户部署个人博客的GitHub仓库的命名必须符合特定要求才行 xxx.github.io

在blog目录下安装hexo-deployer-git插件

cnpm install --save hexo-deployer-git

修改插件配置

_config.yml

​```sh
ls
vim _config.yml
## 定位到 # Deployment 下做配置
deploy:
  type:git
  repo:https://github.com/xxx/xxx.github.io.git
  branch:master

部署到远端

hexo d    #部署到远端仓库（输入账号、密码）

8. 定义博客主题

https://github.com/litten/hexo-theme-yilia

git clone https://github.com/litten/hexo-theme-yilia themes/yilia

打开配置文件

vim _config.md
## 定位到theme
theme: yilia

修改了配置之后，需要执行以下操作

hexo clean
hexo g
hexo s

hugo搭建个人博客

Hugo 基于go语言的博客框架

对比hexo，hugo搭建起来更加简单快速

1. 安装hugo

到官网下载并解压到指定路径下

https://github.com/gohugoio/hugo/releases

windows下载后需要将hugo的安装目录配置环境变量Path中

检测是否安装成功

hugo version

2. 用hugo生成博客

hugo new site myblog

3. 下载并设置主题

https://themes.gohugo.io

以安装m10c为例

cd myblog
git clone https://github.com/jpescador/hugo-future-imperfect.git
## 在配置文件中修改样式配置
#cd config.coml
#theme = "hugo-future-imperfect"
cd themes #检查

4. 在本地启动博客

hugo server -t hugo-future-imperfect --buildDrafts

浏览器打开对于网址检测是否启动成功

4. 创建一篇文章

hugo new post/blog.md

在myblog目录下进入content文件夹

cd content
cd post
vim blog.md #或者在本地通过其他软件进行编写
# ctrl+c 退出运行 执行以下语句重启博客
hugo server -t hugo-future-imperfect --buildDrafts

5. 将个人博客部署到远端服务器

在github创建repo

（https://github.com/Sequin-hdd/Sequin-hdd.github.io）

hugo --theme hugo-future-imperfect --baseUrl="https://xxx.github.io/" --buildDrafts

执行完上面的语句会生成public文件

在public文件夹执行以下操作

cd public
git init
git add .
git commit -m "我的hugo博客 第一次提交"
git remote add origin https://github.com/xxx/xxx.github.io.git
git push -u origin master
